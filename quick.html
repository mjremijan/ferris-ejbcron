Quick Start

Download

Download the ferris-ejbcron-1.0.0.0.war application.

Download Payara Micro

XML Configuration

Create an XML configuration document with the schedules and commands you want to run. See the ScheduleExpression API for more details about what the values in the document may be. Here is an example: 

<?xml version="1.0" encoding="UTF-8"?>
<ScheduleExpressions>
    <ScheduleExpression>
        <Second>0</Second>
        <Minute>0</Minute>
        <Hour>15</Hour>
        <Month>*</Month>
        <Year>*</Year>
        <Command>
            <Executable>C:\Program Files\Internet Explorer\iexplore.exe</Executable>
            <Arguments>
                <Argument>http://www.yahoo.com</Argument>
            </Arguments>
        </Command>
    </ScheduleExpression>
    <ScheduleExpression>
        <Second>0</Second>
        <Minute>0</Minute>
        <Hour>20</Hour>
        <Month>*</Month>
        <Year>*</Year>
        <Command>
            <Executable>C:\Program Files\Internet Explorer\iexplore.exe</Executable>
        </Command>
    </ScheduleExpression>
</ScheduleExpressions>

Run

Run the WAR inside Payara Micro with the following command: 

java.exe -Dejbcron_schedule=C:\your.xml -jar payara-micro.jar --deploy ferris-ejbcron-1.0.0.0.war

That's it! your command will be run when scheduled.
